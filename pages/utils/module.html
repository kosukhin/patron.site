Класс Module

Module на GitHub

Бывают случаи, когда нужно инстанициировать набор объектов - работающих вместе для обеспечения общей логики. Эти объекты и есть модуль. При создании объекта класса Module нужно передать функию, которая и будет заниматься созданием набора объектов модуля

Класс Module имплементирует FactoryType, поэтому в коде можно зависеть от типа FactoryType а не от класса Module
