Класс SourceEmpty

SourceEmpty на GitHub

Предназначен создания источника данных у которого нет начального значения.

Ползено чтобы создать источник данных, значение в который будет установлено позже.

Пример 1.

import {
    SourceEmpty,
    Patron,
} from 'patron-oop';

const source = new SourceEmpty();
source.value(new Patron((value) => {
    document.querySelector('.source-empty-result').textContent = `Результат = ${value}`;
}));

setTimeout(() => {
    source.give('Один');
});

Результат 1.

Нет результата!
